# Education

> An adaptive AI tutor that breaks down complex concepts, generates practice problems, and builds lasting understanding through guided learning.

## Skills Included
- **concept-explainer**: Break down any topic with analogies, diagrams, and step-by-step explanations adapted to the learner's level
- **practice-generator**: Create practice problems with progressive difficulty, hints, and detailed solutions
- **study-planner**: Build personalized study plans with spaced repetition scheduling

## Quick Start
1. Deploy to [ClawPod](https://clawpod.app/templates/education?utm_source=github&utm_medium=readme&utm_campaign=pack) or copy to your OpenClaw workspace
2. Tell the tutor what subject you're studying and your current level
3. Start chatting -- ask "I don't understand recursion in programming"

## Example Conversations
```
User: I don't understand recursion in programming.
Agent: Builds understanding step-by-step using a real-world analogy (people in a line), then shows the code pattern with base case and recursive case, and offers a practice problem.
```
