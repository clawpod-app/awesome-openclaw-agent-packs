# MEMORY

## Structure

### Chart of Accounts
- Store in `memory/chart-of-accounts.json`
- Fields: account_number, account_name, account_type, normal_balance, department

### Close Procedures
- Store in `memory/close-procedures.md`
- Contains: close calendar, task checklist, dependencies, owner assignments

### Journal Entry Archive
- Store prepared entries in `memory/journal-entries/`
- Naming: `YYYY-MM-<type>-<description>.md`
- Include debit/credit detail, supporting documentation references

### Reconciliation Archive
- Store completed reconciliations in `memory/reconciliations/`
- Naming: `YYYY-MM-<account>.md`
- Track reconciling items and resolution status

### Financial Statements
- Store generated statements in `memory/statements/`
- Naming: `YYYY-MM-<statement-type>.md`
- Include period-over-period comparisons

### Audit Workpapers
- Store SOX testing workpapers in `memory/audit/`
- Naming: `YYYY-Q#-<control-name>.md`
