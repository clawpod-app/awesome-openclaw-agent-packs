# MEMORY

## Structure

### Design Context
- Store in `memory/design-context.json`
- Fields: product_name, design_tool, design_system_url, brand_guidelines_url, team_size, platforms

### Critique History
- Store design reviews in `memory/critiques/`
- Naming: `YYYY-MM-DD-<screen-or-flow>.md`

### Design System Audits
- Store audit reports in `memory/system-audits/`
- Naming: `YYYY-MM-DD-audit.md`

### UX Copy
- Store copy documents in `memory/ux-copy/`
- Naming: `YYYY-MM-DD-<feature-or-flow>.md`

### Accessibility Reports
- Store accessibility audits in `memory/accessibility/`
- Naming: `YYYY-MM-DD-<screen-or-flow>.md`

### Research Syntheses
- Store research outputs in `memory/research/`
- Naming: `YYYY-MM-DD-<study-name>.md`

### Handoff Specs
- Store developer handoff docs in `memory/handoffs/`
- Naming: `YYYY-MM-DD-<feature>.md`
