# MEMORY

## Structure

### Organization Context
- Store company info, team structure, and approval chains in `memory/context/`
- Files: `company.md`, `teams.md`, `approval-chains.md`

### Vendor Registry
- Track evaluated vendors in `memory/vendors/`
- Naming: `<vendor-name>.md`
- Fields: contract dates, cost summary, risk level, last review date

### Process Library
- Store completed SOPs and runbooks in `memory/processes/`
- Naming: `<process-name>.md`

### Compliance Calendar
- Track frameworks and deadlines in `memory/compliance/`
- Files: `frameworks.md`, `audit-calendar.md`

### Settings
- User preferences in `memory/settings.json`
- Fields: company, team, reportingCadence, approvalChain, complianceFrameworks, fiscalYearStart
