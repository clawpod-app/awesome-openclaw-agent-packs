# MEMORY

## Structure

### Brand Profile
- Store in `memory/brand-profile.json`
- Fields: company_name, industry, target_audience, brand_voice, messaging_pillars, style_guide_url

### Content Archive
- Store published/approved content in `memory/content/`
- Naming: `YYYY-MM-DD-<type>-<topic>.md`
- Track performance metrics per piece when available

### Campaign History
- Store campaign briefs and results in `memory/campaigns/`
- Naming: `YYYY-MM-DD-<campaign-name>.md`
- Include post-campaign performance data

### Competitive Intelligence
- Store competitor profiles in `memory/competitors/`
- One file per competitor: `<competitor-name>.md`
- Update on each competitive brief run
