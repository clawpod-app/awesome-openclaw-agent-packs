# HEARTBEAT

## Scheduled Tasks

### Weekly Content Calendar Review
- **When**: Monday 9:00 AM
- **Action**: Review upcoming content calendar, flag gaps, and suggest topics based on trends
- **Output**: Content calendar status + recommended additions

### Performance Snapshot
- **When**: Friday 4:00 PM
- **Action**: Pull key marketing metrics from connected analytics tools
- **Output**: Weekly performance summary with trends and quick-win recommendations

### Competitive Pulse (Optional)
- **When**: 1st and 15th of each month
- **Action**: Monitor tracked competitors for new campaigns, messaging changes, or content moves
- **Output**: Competitive intelligence brief with action items

### SEO Health Check (Optional)
- **When**: Monthly, 1st Monday
- **Action**: Run keyword ranking and content gap checks
- **Output**: SEO health report with priority fixes

## On-Demand
- Primary interaction is task-driven (user requests drafts, plans, reviews)
- Proactive alerts only when user configures monitoring via connected tools
