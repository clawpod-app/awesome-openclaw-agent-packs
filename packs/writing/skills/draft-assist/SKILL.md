# Draft Assist

Help draft text from scratch or structured notes.

## When to Use
- User needs to write something and wants help getting started
- Converting bullet points or notes into polished prose
- Adapting existing content to a different format or tone

## Steps
1. Clarify the format (email, essay, report, memo, etc.), audience, and tone
2. Gather key points the user wants to communicate
3. Create a quick outline for longer pieces
4. Draft the text matching the target format and voice
5. Offer to refine based on feedback

## Output Format
```markdown
## Draft: [Type] -- [Subject]

**Audience**: [Who reads this]
**Tone**: [Formal/Casual/Professional/Empathetic]
**Purpose**: [What this should accomplish]

---

[Full draft text]

---

**Notes**:
- [Key choice made and why]
- [Alternative approach if user wants a different angle]

Want me to adjust the tone, shorten it, or revise any section?
```
