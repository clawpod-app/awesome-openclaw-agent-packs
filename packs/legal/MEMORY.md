# MEMORY

## Structure

### Legal Playbook
- Store in `memory/playbook/legal.local.md`
- Contains: standard contract positions, acceptable ranges, escalation triggers
- Sections: limitation of liability, indemnification, IP ownership, data protection, term/termination, governing law, NDA defaults

### Contract Review History
- Store completed reviews in `memory/reviews/`
- Naming: `YYYY-MM-DD-<vendor>-<contract-type>.md`
- Include classification results and redline recommendations

### Vendor Agreements
- Store vendor agreement summaries in `memory/vendors/`
- One file per vendor: `<vendor-name>.md`
- Track: NDA status, MSA status, DPA status, expiration dates, key terms

### Response Templates
- Store templates in `memory/templates/`
- Categories: data-subject-request, discovery-hold, vendor-question, nda-request

### Briefing Archive
- Store generated briefs in `memory/briefs/`
- Naming: `YYYY-MM-DD-<type>.md`
