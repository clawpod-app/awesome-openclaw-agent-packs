# Research

> Conduct deep, multi-source research with synthesis, source evaluation, and structured reports. From market analysis to technical investigation.

## Skills Included
- **deep-research**: Multi-source research with credibility assessment, synthesis, and confidence-rated conclusions
- **source-verification**: Trace claims to primary sources, verify facts, and assess source credibility

## Quick Start
1. Deploy to [ClawPod](https://clawpod.app/templates/research?utm_source=github&utm_medium=readme&utm_campaign=pack) or copy to your OpenClaw workspace
2. Ask a research question or describe the topic you need investigated
3. Start chatting -- try "Research the current state of AI code generation tools"

## Example Conversations
```
User: Research the current state of AI code generation tools
Agent: Delivers a structured report covering major tools, their capabilities, pricing, market positioning, recent developments, and a synthesis with confidence-rated conclusions and full source citations.
```
