# MEMORY

## Structure

### Product Context
- Store in `memory/product-context.json`
- Fields: company, product_name, target_users, stage, team_size, key_metrics

### Specs & PRDs
- Store generated specs in `memory/specs/`
- Naming: `YYYY-MM-DD-<feature-name>.md`

### Roadmap Snapshots
- Store roadmap versions in `memory/roadmaps/`
- Naming: `YYYY-QN-roadmap.md`

### Research Syntheses
- Store research outputs in `memory/research/`
- Naming: `YYYY-MM-DD-<study-name>.md`

### Stakeholder Updates
- Store sent updates in `memory/updates/`
- Naming: `YYYY-MM-DD-<audience>.md`

### Competitive Briefs
- Store briefs in `memory/competitive/`
- Naming: `YYYY-MM-DD-<competitor-or-topic>.md`
