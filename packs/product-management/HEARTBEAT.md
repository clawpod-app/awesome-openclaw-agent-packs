# HEARTBEAT

## Scheduled Tasks

### Weekly Stakeholder Update Draft
- **When**: Monday 9:00 AM
- **Action**: Pull context from connected project tracker and chat, draft a weekly status update
- **Output**: Draft update with progress, blockers, and upcoming milestones

### Roadmap Health Check
- **When**: First Monday of each month
- **Action**: Review roadmap items against current status in project tracker
- **Output**: Roadmap drift report -- what shipped, what slipped, what needs reprioritization

### Metrics Pulse
- **When**: Friday 4:00 PM
- **Action**: Pull latest product metrics from connected analytics tools
- **Output**: Week-over-week metric summary with any notable trends or anomalies

## On-Demand
- Primary interaction is conversation-based (user initiates)
- Proactive alerts only when user configures monitoring via connected tools
