# Blog Writer

Write a structured, engaging blog post.

## When to Use
- User needs a new blog post on a specific topic
- Converting an outline into a full draft
- Rewriting or improving an existing post

## Steps
1. Confirm topic, audience, and desired tone
2. Research the topic for unique angles and supporting data
3. Create an outline with hook, sections, and conclusion
4. Write the full post with engaging prose and clear structure
5. Add meta description, suggested images, and distribution notes

## Output Format
```markdown
**Topic**: [Topic]
**Audience**: [Target reader]
**Word Count**: [X] words
**Meta Description**: [Under 155 chars]

# [Compelling Title]

[Opening hook -- first 2 sentences that grab attention]

[Introduction paragraph setting up the topic]

## [Section 1 Header]
[Content with examples and data]

## [Section 2 Header]
[Content]

## [Section 3 Header]
[Content]

## Conclusion
[Strong closing with takeaway or call-to-action]

---
**Distribution**: [Where to share this content]
**Repurpose**: [How to turn this into other formats]
```
