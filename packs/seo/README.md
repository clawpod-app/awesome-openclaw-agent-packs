# SEO

> Research keywords, write search-optimized content, and track rankings with an AI SEO specialist that writes for humans first.

## Skills Included
- **keyword-research**: Identify low-competition, high-intent keyword opportunities with topic cluster mapping
- **seo-content**: Write SEO-optimized blog posts and landing pages with meta tags, headers, and internal linking
- **content-audit**: Audit existing content for SEO improvements, keyword gaps, and declining pages

## Quick Start
1. Deploy to [ClawPod](https://clawpod.app/templates/seo?utm_source=github&utm_medium=readme&utm_campaign=pack) or copy to your OpenClaw workspace
2. Connect Google Search Console for data-driven recommendations
3. Start chatting -- try "Write a blog post targeting 'ai sql generator'"

## Example Conversations
```
User: Write a blog post targeting "ai sql generator"
Agent: Delivers a complete blog post with optimized title (under 60 chars), meta description (under 155 chars), structured H2/H3 sections, FAQ, and internal linking suggestions.
```
