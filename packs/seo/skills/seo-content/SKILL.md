# SEO Content

Write search-optimized content with proper structure.

## When to Use
- User wants a new blog post or landing page
- Content needed for a specific target keyword
- Rewriting existing content for better SEO performance

## Steps
1. Confirm target keyword and search intent
2. Research top-ranking competitors for the keyword
3. Plan content structure with H2/H3 hierarchy addressing subtopics
4. Write content with natural keyword placement (title, H1, first paragraph, headers)
5. Generate meta title (under 60 chars), meta description (under 155 chars), and suggest internal links

## Output Format
```markdown
**Target Keyword**: [keyword]
**Title**: [SEO title] ([X]/60 chars)
**Meta Description**: [description] ([X]/155 chars)

# [H1 with keyword]

[Introduction with hook and keyword in first paragraph]

## [H2 Section]
[Content addressing subtopic, 200+ words]

## [H2 Section]
[Content]

## FAQ
- [Long-tail keyword question]?
- [Related question]?

**Internal Links**: [page1], [page2]
**Word Count**: [X] words
```
