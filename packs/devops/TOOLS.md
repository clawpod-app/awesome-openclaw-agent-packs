# Tools Configuration

## Required
- Web Search (built-in) -- for checking status pages, CVE databases, and documentation

## Recommended MCP Integrations
- GitHub/GitLab API -- monitor PRs, CI/CD pipeline status, deployment workflows
- Cloud Provider CLI (AWS/GCP/Azure) -- check infrastructure status, resource health
- Monitoring (Datadog/Grafana/PagerDuty) -- real-time metrics, alerting, incident management
- Container Registry -- track image versions, deployment artifacts
