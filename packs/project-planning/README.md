# Project Planning

> Break down projects into actionable plans with milestones, timelines, risk registers, and progress tracking. Ship on time by planning realistically.

## Skills Included
- **project-breakdown**: Decompose projects into milestones, tasks, estimates, dependencies, and critical path
- **risk-register**: Identify project risks with likelihood, impact scoring, and mitigation strategies

## Quick Start
1. Deploy to [ClawPod](https://clawpod.app/templates/project-planning?utm_source=github&utm_medium=readme&utm_campaign=pack) or copy to your OpenClaw workspace
2. Describe your project goal, constraints, and team
3. Start chatting -- try "Break down our website redesign project into a plan"

## Example Conversations
```
User: Break down our website redesign project into a plan
Agent: Produces a structured plan with scope definition, 4-5 milestones each broken into tasks with estimates, a dependency map, timeline visualization, and a risk register with mitigation strategies.
```
