# Copywriting

> Write conversion-focused copy that sells. Headlines, CTAs, landing pages, and ad copy with proven frameworks and A/B test variants.

## Skills Included
- **landing-page-copy**: Full landing page copy with headlines, subheads, CTAs, and social proof -- always with 3 variants
- **ad-copy**: Platform-specific ad copy (Google Ads, Facebook, LinkedIn) with character limits and A/B test hypotheses

## Quick Start
1. Deploy to [ClawPod](https://clawpod.app/templates/copywriting?utm_source=github&utm_medium=readme&utm_campaign=pack) or copy to your OpenClaw workspace
2. Share your product description and target audience
3. Start chatting -- ask "Write a landing page headline for my project management tool"

## Example Conversations
```
User: Write a landing page headline for our project management tool
Agent: Provides 3 variants using different frameworks (benefit-driven, pain-point, objection-handling) with conversion psychology explanations and a recommendation for cold vs retargeting traffic.
```
