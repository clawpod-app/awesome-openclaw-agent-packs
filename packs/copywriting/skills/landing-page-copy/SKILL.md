# Landing Page Copy

Write complete landing page copy optimized for conversion.

## When to Use
- User needs copy for a new landing page
- User wants to improve existing landing page conversion rates
- User needs headline or CTA variants for testing

## Steps
1. Understand the product, target audience, and conversion goal
2. Identify the best framework (AIDA for awareness, PAS for pain-aware, BAB for solution-aware)
3. Draft 3 headline variants with different angles
4. Write supporting copy: subheads, bullet points, social proof, CTA
5. Explain the conversion psychology behind each variant

## Output Format
```markdown
## Landing Page Copy -- [Product Name]

**Audience**: [Target audience]
**Goal**: [Conversion goal]
**Framework**: [AIDA/PAS/BAB]

### Headlines (3 Variants)
1. "[Headline]" -- [Framework used, why it works]
2. "[Headline]" -- [Framework used, why it works]
3. "[Headline]" -- [Framework used, why it works]

**Recommendation**: #[N] for [audience type] because [reasoning]

### Supporting Copy
- **Subhead**: [Supporting the winning headline]
- **Bullets**: [3-5 benefit-driven bullets]
- **Social Proof**: [Testimonial or stat suggestion]
- **CTA**: "[Button text]" + [supporting microcopy]
```
