# Social Post

Write platform-native social media content.

## When to Use
- User wants to post about a specific topic
- Content calendar slot needs to be filled
- User wants to share a product update, story, or insight

## Steps
1. Confirm topic, target platform, and audience
2. Choose the right format (single post, thread, carousel text)
3. Write a strong hook in the first line
4. Adapt tone and length to the platform
5. Provide 2-3 variants with different angles

## Output Format
```markdown
## [Platform] Post -- [Topic]

### Option A
[Full post text]

### Option B
[Full post text]

### Option C (if applicable)
[Full post text]

**Recommended**: Option [X] because [reasoning]
**Hashtags**: [2-3 relevant hashtags]
**Best Time**: [Suggested posting time]
```
