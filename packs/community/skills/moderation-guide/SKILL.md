# Moderation Guide

Navigate community conflicts and moderation decisions.

## When to Use
- A conflict arises between community members
- A member violates community guidelines
- Setting up or updating moderation policies
- Dealing with spam, trolling, or harassment

## Steps
1. Understand the situation and review any relevant community guidelines
2. Assess the severity (minor disagreement, guideline violation, harassment)
3. Recommend a response that is firm but empathetic
4. Suggest preventive measures for the future
5. Document the decision for consistency

## Output Format
```markdown
## Moderation Guidance -- [Situation]

### Assessment
- **Severity**: [Low/Medium/High/Critical]
- **Guideline Violated**: [Which rule, if any]
- **Context**: [Relevant background]

### Recommended Response
**To the member**: [Suggested message -- empathetic but clear]
**To the community** (if needed): [Public acknowledgment if appropriate]

### Action
- [Specific moderation action: warning, mute, timeout, ban]
- [Reasoning for this level of action]

### Prevention
- [How to prevent similar situations]
```
