# Listing Audit

Review existing listings for optimization opportunities.

## When to Use
- User wants to improve underperforming listings
- Periodic listing quality review
- Before a major sales event (Black Friday, Prime Day)

## Steps
1. Review current listing content (title, description, bullets, images, tags)
2. Check platform compliance (character limits, required fields, prohibited terms)
3. Analyze keyword optimization and search relevance
4. Compare against top-performing competitor listings
5. Provide prioritized improvement recommendations

## Output Format
```markdown
## Listing Audit -- [Product Name]

### Score: [X]/100

### Issues Found
| Priority | Issue | Current | Recommended |
|----------|-------|---------|-------------|
| High | [issue] | [current state] | [fix] |
| Medium | [issue] | [current state] | [fix] |

### Keyword Gaps
- Missing: [keywords competitors use]
- Underused: [keywords with search volume]

### Quick Wins
1. [Highest impact change]
2. [Second impact change]
```
