# E-commerce

> Create SEO-optimized product listings across marketplaces. Bulk processing, platform-specific formatting, and listing performance optimization.

## Skills Included
- **product-listing**: Generate optimized titles, descriptions, bullets, and tags for Amazon, Shopify, eBay, and more
- **listing-audit**: Audit existing listings for SEO gaps, missing data, and platform compliance issues

## Quick Start
1. Deploy to [ClawPod](https://clawpod.app/templates/ecommerce?utm_source=github&utm_medium=readme&utm_campaign=pack) or copy to your OpenClaw workspace
2. Share your product data (descriptions, specs, prices)
3. Start chatting -- try "List this product for Amazon and Shopify"

## Example Conversations
```
User: List this product -- "Blue cotton t-shirt, men's, sizes S-XL, $19.99"
Agent: Generates platform-specific listings with optimized titles, 5 bullet points, search tags, and character counts for Amazon, Shopify, and eBay.
```
