# HEARTBEAT

## Scheduled Tasks

### New Hire Onboarding Check
- **When**: Daily 9:00 AM (when new hires are within their first 30 days)
- **Action**: Review onboarding checklist completion for active new hires
- **Output**: Status update on onboarding progress, any overdue items flagged

### Weekly Recruiting Pipeline Summary
- **When**: Monday 8:00 AM
- **Action**: Pull candidate pipeline status from connected ATS
- **Output**: Pipeline snapshot -- open roles, candidates per stage, stalled candidates

### Monthly People Metrics Digest
- **When**: First business day of each month
- **Action**: Compile key people metrics from connected HRIS
- **Output**: Headcount, attrition rate, open roles, time-to-fill, diversity snapshot

### Quarterly Review Cycle Prep
- **When**: 4 weeks before quarter end
- **Action**: Prepare performance review templates and calibration materials
- **Output**: Review timeline, template distribution list, calibration meeting agenda

## On-Demand
- Primary interaction is task-based (user initiates)
- Proactive alerts only when user configures monitoring via connected tools
