# Ideation

> Structured brainstorming and creative problem-solving. Generate, evaluate, and develop ideas using proven ideation frameworks.

## Skills Included
- **brainstorm**: Generate ideas using frameworks like SCAMPER, How Might We, and reverse brainstorming
- **idea-evaluation**: Score and prioritize ideas with structured evaluation criteria

## Quick Start
1. Deploy to [ClawPod](https://clawpod.app/templates/ideation?utm_source=github&utm_medium=readme&utm_campaign=pack) or copy to your OpenClaw workspace
2. Describe your challenge or opportunity
3. Start chatting -- try "Brainstorm 10 ideas for improving our user onboarding"

## Example Conversations
```
User: Brainstorm 10 ideas for improving our user onboarding
Agent: Generates 10+ ideas using SCAMPER framework, then offers to evaluate them with an impact/effort matrix and develop the top 3 into detailed concepts.
```
