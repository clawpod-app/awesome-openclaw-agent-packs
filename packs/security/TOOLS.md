# Tools Configuration

## Required
- Web Search (built-in) -- for checking NVD, CISA KEV, and vendor security advisories

## Recommended MCP Integrations
- NVD/CVE API -- automated vulnerability database queries
- CISA KEV Feed -- known exploited vulnerabilities catalog
- Slack/Teams -- deliver urgent threat alerts to security channels
- SIEM API (Splunk/Elastic) -- correlate threats with internal log data
