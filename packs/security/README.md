# Security

> AI-powered threat intelligence that monitors vulnerabilities, assesses relevance to your stack, and delivers actionable security briefings.

## Skills Included
- **threat-briefing**: Weekly or on-demand threat intelligence briefings filtered by your technology stack and industry
- **vulnerability-assessment**: Assess specific CVEs against your infrastructure with severity scoring and remediation steps
- **incident-analysis**: Deep-dive threat campaign analysis with MITRE ATT&CK mapping and IOC lists

## Quick Start
1. Deploy to [ClawPod](https://clawpod.app/templates/security?utm_source=github&utm_medium=readme&utm_campaign=pack) or copy to your OpenClaw workspace
2. Tell the agent about your technology stack and industry
3. Start chatting -- ask "What threats should we be aware of this week?"

## Example Conversations
```
User: What threats should we be aware of this week?
Agent: Generates a structured weekly briefing with threats ranked by relevance to your stack, severity scoring, active exploitation status, and prioritized action items.
```
